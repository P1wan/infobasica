Informações Gerais

Glossário de Termos de Navegação

  • Link: Um elemento clicável que leva a outra página ou seção.
  • Botão: Um componente usado para interações, como enviar um formulário.
  • Caixa de Seleção: Um campo onde você pode marcar ou desmarcar uma opção.
  • HTTPS: Protocolo de navegação seguro, indicado pelo cadeado na barra de endereço.

Atalhos Básicos do Windows

  • Ctrl + C: Copiar
  • Ctrl + V: Colar
  • Ctrl + X: Recortar
